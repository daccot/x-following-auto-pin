# X Following Auto Pin v0.4.2 i18n

This build is based on v0.4.1 Stable Logging.

## Behavior

- Reverted to v0.4.1 behavior
- Keeps Following pin and Latest pin logic unchanged
- Adds UI language selection for the options page
- Supports Japanese and English messages
- Default language is selected from the browser language
  - Japanese browser: Japanese
  - Other languages: English

## Settings

- Following pin
- Latest / chronological pin
- Light logging
- UI language: Auto / Japanese / English
