# Privacy Policy

X Following Auto Pin does not collect, transmit, sell, or share personal information.

## Data collection

This extension does not collect user data.

## Local storage

The extension uses Chrome storage APIs only to save local extension settings, such as:

- Whether Following pin is enabled
- Whether Latest / chronological pin is enabled
- Whether light logging is enabled
- UI language preference, if available

These settings remain in the user's browser.

## Network transmission

The extension does not transmit data to any external server.

## Remote code

The extension does not use remote code. All scripts are packaged with the extension.

## Website access

The extension runs only on:

- `x.com`
- `twitter.com`

It accesses the page DOM only to detect and select existing X timeline UI controls, such as the For you / Following tabs and the Popular / Latest sort menu.

## Contact

For questions, please open an issue in this repository.
